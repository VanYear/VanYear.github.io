unity基础学习13-UI

发表于2024-10-16|更新于2024-10-17|unity
|阅读量:

1.画布

Canvas。在画布上添加均是作为子物体存在。渲染模式：覆盖使UI始终在最上层，摄像机则会使通过摄像机渲染同时画布始终面向摄像机，世界空间则将画布也当作3D物体

UI缩放，可根据屏幕显示或者恒定的分辨率

2.图像

图像的位置是相对于锚点。将锚点四个点分开，四个点有相对于整个屏幕的比例，因此在不同屏幕上锚点依然会占相应的比例，锚点一般会直接放置在屏幕四个角上，而图像的位置及缩放是相对于锚点的所以当屏幕和锚点都改变时，图像也会随之改变，即可实现UI在不同屏幕的适配。

同时图像跟锚点的位置是根据图片的轴心点计算的，包括图像的缩放和选择均依赖于轴心点

3.文本

旧版：可直接输入并设置字体、粗体等，文本输入时也可使用类似html方式对部分文本进行设置

新版：采用TMP，可以更准确的设置字体

4.按钮

包含文本和图像组件。可以设置按钮交互，包括点击以及移动到按钮上时颜色、图片的变化。也可设置导航，使用按键选择按钮。在按钮上添加事件需添加物体再选择挂载到该物体上的组件和脚本中的方法。

5.输入框

包含占位文本以及用户输入文本，可设置用户输入的内容限制。在输入值改变时和提交时都可使用事件，方法与按钮相同

注：新版为TMPro，旧版为UI

6.选项与下拉框

选项包含背景图片和文本。设置单选：其中一个选项添加Toggle Group组件，将该组件拖入多个选项，使多个选项绑定一个Toggle Group。下拉框与文本和选项类似。下拉框中的每一个选项都是由Template模板生成。添加图像：要在Template-Viewport-Content-Item中添加图像组件，并在项图像和标题图像中添加创建的Image。

7.滑动条、滚动条与滚动视图

滑动条：常用于音量调节和血条等，主要用Value值

滚动条：显示浏览位置

滚动视图：用以显示更多内容，自带横向和纵向滚动条。真正的内容加到content

8.面板

将控件放在面板上作为面板的子物体，子控件和面板的锚点绑定，面板再和画布的锚点绑定，便于分类移动和适配UI

9.其他常用组件

遮罩Mask，超出父物体范围的内容不显示

Content Size Fitter，文本框根据文本内容自动进行适配

Vertical Layout Group，垂直布局，多个子物体设置列表

Horizontal Layout Group，水平布局

Grid Layout Group，网格布局，和滚动视图组合可制作背包等功能
